# Project-Root
This is my project on training 1D  Convolutional Neural Network for predicting the irregularities in breathing during sleeping. 
My first task was to create clear visualization of the data for each participant, making it easier to interpret patterns and irregularities.
My second task was to apply digital filtering techniques to retain only the relevant breathing frequency range.
My third task was to train a 1D CNN model to classify the breathing irregularities based on the labeled 30 second window.
